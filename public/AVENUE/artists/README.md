1. Baixe as fotos oficiais (Instagram, site, assessoria)
2. Salve nesta pasta com os nomes exatos:

| Arquivo | Artista |
|---------|---------|
| `tutto.jpg` ou `tutto.png` | MC Tuto |
| `chefin.jpg` ou `chefin.png` | Chefin |
| `meno-k.jpg` ou `meno-k.png` | MC Meno K |
| `ig.jpg` ou `ig.png` | MC IG |
